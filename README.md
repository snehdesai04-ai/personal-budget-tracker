# 💰 Personal Budget Tracker

A web-based personal finance tracker that allows users to manage their income and expenses in real time.

Built by **Sneh Desai** — Computer Science Student & SaskTel Employee, University of Regina

---

## 🌐 Live Demo

👉 [Click here to view the live project](https://snehdesai04-ai.github.io/personal-budget-tracker-v2/)

---

## 📸 Features

- ➕ Add income and expense transactions
- 🗂️ Categorize transactions (Salary, Food, Rent, Education, etc.)
- 🔍 Filter by All / Income / Expenses
- 💵 Real-time balance, total income, and total expense calculation
- 🗑️ Delete individual transactions
- 📱 Responsive design — works on desktop and mobile

## 📁 Project Structure

```
personal-budget-tracker/
├── index.html    → HTML structure (the skeleton)
├── style.css     → CSS styling (the design)
├── script.js     → JavaScript logic (the brain)
└── schema.sql    → SQL database schema (the data layer)
```

---

## 🗄️ SQL Schema Highlights

The `schema.sql` file includes:
- `CREATE TABLE` — defines the transactions table
- `INSERT` — adds new transactions
- `SELECT` — retrieves and filters data
- `SUM / GROUP BY` — calculates totals and summaries
- `DELETE / UPDATE` — manages existing records

---

## 🚀 How to Run Locally

1. Clone or download this repository
2. Open the folder in **VS Code**
3. Right-click `index.html` → **Open with Live Server**
4. The app opens in your browser automatically
